Agricultura Moderna

Identificar pragas

Devido a grande quantidade de pragas a identificação torna-se um trabalho árduo

Quantidade de cultivos

A quantia de cultivos torna inviável o mapeamento humano das pragas sem que haja perdas significativas

Tempo de reação

As pragas podem atacar a qualquer momento e nem sempre você estará preparado

Solução

Monitoramento

Câmeras especiais são utilizadas no monitoramento da lavoura

Reação rápida

O sistema reage automaticamente após a detecção de uma praga

Inteligência artificial

Um modelo treinado de inteligência artificial é usado na detecção

Monitoramento por câmera

O sistema utiliza monitoramento por câmeras em conjunto com sensores terrestres para identificar e enviar imagens das pragas em caso de identificação

Inteligência artificial

O sistema possuirá uma inteligência artificial em seu backend que irá receber requests com as imagens das pragas para retornar o nome da praga e acionar automaticamente o pesticida adequado para aquela praga conforme o modelo foi treinado.

Banco de dados

Fotos capturadas pelo sistema com a identificação da praga são enviadas pelo backend para um banco de dados registrando a quantidade de ocorrências

Controle

O usuário poderá escolher entre consumir os dados e controlar o sistema através de um aplicativo mobile, ou um programa instalado no computador
